CloseoutRequestIntentBuilder

public class CloseoutRequestIntentBuilder extends BaseIntentBuilder

Use the CloseoutRequestIntentBuilder to build an intent for a merchant to closeout transactions. Before a closeout can be successful, the merchant must be approved for manual closeout before using, as well as open payments must have a tip amount set. Optionally, through TipOptions, you can choose to set a zero amount for all remaining open payments, to leave preauths open, and for Clover to handle tips.
