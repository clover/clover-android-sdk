RemotePaymentsAPIConnector

public final class RemotePaymentsAPIConnector

A connector that enables device requests (e.g. Read Card), using Android Payments API created intents on Customer Facing devices. This will ONLY work on remote devices that are tethered to a Merchant Facing Clover device.
