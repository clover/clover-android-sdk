CashDrawers

public class CashDrawers

This class provides functions for working with cash drawers. Specifically, it allows clients to list all connected cash drawers from known hardware types. More types of cash drawers may be added over time.

This class differs from the com.clover.sdk.v1.printer.CashDrawer class in that it abstracts any hardware cash drawer, where com.clover.sdk.v1.printer.CashDrawer only works with cash drawers connected to configured Clover printers. Old implementations of this class used a hardcoded list of supported cash drawers, this newer implementation allows Clover to add support for new cash drawer types as they become available.
