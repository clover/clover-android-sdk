CloverCFPActivityHelper

public class CloverCFPActivityHelper extends ExitTouchHandler

Provides helper methods for handling Activities on Clover devices. If you use CloverCFPLoyaltyHelper, it will use this class internally to interact with the Activity.

In particular, attaches the 4-finger exit handler (ExitHandler) to the activity.
