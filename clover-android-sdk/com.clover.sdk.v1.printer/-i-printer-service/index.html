IPrinterService

public interface IPrinterService implements IInterface

An interface for adding, listing, and removing Clover printers. To print to a Clover printer, use the classes in the com.clover.sdk.v1.printer.job package. To open a cash drawer connected to a Clover printer, use the com.clover.sdk.v1.printer.CashDrawer class. To interact with the Clover print queue, use the com.clover.sdk.v1.printer.PrintQueue class. The Printer service is a bound AIDL service. Bind to this service as follows, 


Intent serviceIntent = new Intent(PrinterIntent.ACTION_PRINTER_SERVICE);
serviceIntent.putExtra(PrinterIntent.EXTRA_ACCOUNT, CloverAccount.getAccount(context));
serviceIntent.putExtra(PrinterIntent.EXTRA_VERSION, 1);
context.bindService(serviceIntent);
