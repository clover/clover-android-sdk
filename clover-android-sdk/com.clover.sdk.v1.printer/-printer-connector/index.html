PrinterConnector

public class PrinterConnector extends ServiceConnector<S>

A class that encapsulates interaction with com.clover.sdk.v1.printer.IPrinterService. This class automatically binds and provides both synchronous and asynchronous service method invocation.

Clients of this class may optionally call connect to force pre-binding to the underlying service, and must call disconnect when finished interacting with the underlying service. The preferred methods in this class are synchronous methods which do not use callbacks and should be invoked on a non-main thread.
