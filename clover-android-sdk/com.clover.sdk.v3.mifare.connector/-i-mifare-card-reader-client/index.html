IMifareCardReaderClient

public interface IMifareCardReaderClient

Mifare Client to read mifare and mobile card read operations. This will help in reading mifare card operations. Please make sure to finish any card reading/writing operation with in 30 seconds. If you couldn't finish any operation with in 30 seconds, please invoke connect again.
