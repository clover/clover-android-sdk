LineItemEventContract

public final class LineItemEventContract

This class defines the contract for accessing the line item events content provider. This content provider does not support updates or deletes. To insert a single line item event, use the content provider call method METHOD_INSERT, or the method insert. To insert several line item events at once, prefer the content provider call method METHOD_BULK_INSERT, or the method bulkInsert.
