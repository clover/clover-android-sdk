OrderContract

public final class OrderContract

The contract between the orders provider and apps. For more detailed information about orders please see IOrderServiceV3_1.

The content provider for orders may be used directly by apps wishing to use a android.database.Cursor to navigate a large orders database. Specifically using the contract allows an app to discover all the recent orders (via query). Read Content provider basics for an overview of how to interact with a content provider. In this case only methods which read data such as query will work. Methods which write data will be rejected. Using this contract requires the client app have ORDERS_R permission. To mutate orders use through the wrapper class com.clover.sdk.v3.order.OrderConnector. The orders provider which fulfills this contract supplies only recent orders (see OrderSummary). Older orders can be fetched via the OrderConnector.
