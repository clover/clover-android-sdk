public class OrderV3Connector extends ServiceConnector<S>

Deprecated

Service connector for IOrderService. Please see that class for documentation on the RPC methods.

Deprecated

Please use the OrderConnector instead, it offers all the same functionality but is designed to handle large orders (over 1MB) that cause this API to fail. New methods may not be added to this class going forward.
