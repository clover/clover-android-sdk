CloverTouchPointActivity

public abstract class CloverTouchPointActivity extends Activity implements CFPSessionListener

A convenience base class for a TouchPoint Custom Activity that provides capabilities to simplify data flow in, and out of, a custom activity

Note: This class is NOT strictly required for an activity to be launched as a touch point activity. CloverCFPActivityHelper, CloverCFPLoyaltyHelper and SessionConnector can be used to get the same capabilities without extending this activity. However, extending this class is recommended, as there are other included behaviors which aid in proper function within the Clover platform.
