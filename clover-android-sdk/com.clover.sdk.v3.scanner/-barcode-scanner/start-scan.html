public void startScan()

Open the barcode scanning app and begin scanning. See BarcodeResult for details on receiving the results. May block momentarily, prefer executeStartScan.

public boolean startScan(Bundle extras)

Open the barcode scanning app and begin scanning. See BarcodeResult for details on receiving the results. May block momentarily, prefer executeStartScan.

Return

true if the scanning app started (it would be very unusual for it to fail)

Parameters

extras

optional parameters, currently supports:

  • EXTRA_SCANNER_FACING - if there are multiple cameras which one to scan from, see getAvailable
  • >
  • EXTRA_SHOW_PREVIEW - whether scanner preview video will be shown, default is true. The preview window cannot be hidden on Clover Flex 1 (C401) devices.
  • EXTRA_LED_ON - whether LED will be on (selected devices only), default is false
  • EXTRA_SCAN_QR_CODE - whether QR codes will be scanned, default is true
  • EXTRA_SCAN_1D_CODE - whether 1D codes will be scanned, default is true
  • EXTRA_TRANSMIT_UPC_A_CHECK_DIGIT - whether UPC-A check digit can be transmitted, default is true
  • EXTRA_TRANSMIT_UPC_A_PREAMBLE - whether UPC-A preamble can be transmitted, default is true
  • EXTRA_SHOW_CLOSE_BUTTON - whether scanner preview will have a 'close' button, default is true
  • EXTRA_SHOW_LED_BUTTON - whether scanner preview will have an LED toggle, default is true
