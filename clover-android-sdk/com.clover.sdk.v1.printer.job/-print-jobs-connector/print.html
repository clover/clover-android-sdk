@Nullable()
public String print(@NonNull() PrintJob printJob)

Print a PrintJob to any configured suitable printer. A suitable printer is one where the printer's category (getCategory matches the print job's category (getPrinterCategory. If no suitable printer is configured the job is not printed and null is returned. If there are multiple suitable configured printers the order of selection is not defined. The selection is not stable; it may be different for each invocations of this method. If it is preferable to ensure a suitable printer is configured before printing call getPrinters passing getPrinterCategory as the argument, and pass the explicit printer argument to print instead of this method. This method is a convenience. The caller is not required to retrieve and evaluate the list of configured printers first and select one, possibly involving the user. Most merchants have at most one printer of each type (receipt, order, label, fiscal) configured. This however is not guaranteed. This method is equivalent to calling print with printToAny set to true. The actual value of printToAny is not considered here.

Return

a String, the print job ID or null if the print job was not queued.

Parameters

printJob

a PrintJob, the print job to print.
