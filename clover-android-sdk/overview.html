The Clover SDK for Android facilitates development of applications on top of the Clover platform. This includes services for interacting with merchants, their orders, customers, employees, and inventory. This SDK can also be used for sending and receiving push notifications.

To access merchant information, employee, customer, inventory, and order data, bound AIDL services are available.

  • {@link com.clover.sdk.v1.merchant.IMerchantService}
  • {@link com.clover.sdk.v1.employee.IEmployeeService}
  • {@link com.clover.sdk.v1.customer.ICustomerService}
  • {@link com.clover.sdk.v1.inventory.IInventoryService}
  • {@link com.clover.sdk.v1.order.IOrderService}

For more information about bound services, refer to the Android documentation: Bound Services .

To bind to any Clover service, a Clover {@link android.accounts.Account} is required. To obtain a Clover account object, the {@link com.clover.sdk.util.CloverAccount} helper class may be used. To allow the user to select the Clover account from those present on the device, 

    Intent intent = AccountManager.newChooseAccountIntent(
        null,
        null,
        new String[]{CloverAccount.CLOVER_ACCOUNT_TYPE},
        false,
        null,
        null,
        null,
        null);
    startActivityForResult(intent, REQUEST_ACCOUNT);
See {@link android.accounts.AccountManager#newChooseAccountIntent(Account, ArrayList, String[], boolean, String, String, String[], Bundle)} for more information.

Each service has a matching {@code *Intent} class that defines the actions and extras required to bind. These classes are as follows,

  • {@link com.clover.sdk.v1.merchant.MerchantIntent}
  • {@link com.clover.sdk.v1.employee.EmployeeIntent}
  • {@link com.clover.sdk.v1.customer.CustomerIntent}
  • {@link com.clover.sdk.v1.inventory.InventoryIntent}
  • {@link com.clover.sdk.v1.order.OrderIntent}

Services can also be interacted with by using their respective service connector class. A service connector automatically binds and provides convenience methods for calling service methods asynchronously. The following service connectors are available, mirroring each bound service,

  • {@link com.clover.sdk.v1.merchant.MerchantConnector}
  • {@link com.clover.sdk.v1.employee.EmployeeConnector}
  • {@link com.clover.sdk.v1.customer.CustomerConnector}
  • {@link com.clover.sdk.v1.inventory.InventoryConnector}
  • {@link com.clover.sdk.v1.order.OrderConnector}
