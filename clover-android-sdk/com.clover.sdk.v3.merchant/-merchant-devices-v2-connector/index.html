MerchantDevicesV2Connector

public class MerchantDevicesV2Connector extends CallConnector

Access information about devices associated with this merchant. This class contains convenience wrappers for accessing the data stored in the content provider at URI CONTENT_URI.

As the total number of devices for a merchant can be large, use MerchantDevicesV2Contract.Device to query for all (or a subset) of the merchant's devices. All of the methods in this class are blocking. They should not be called on the main thread.
