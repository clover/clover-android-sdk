FdParcelable

public class FdParcelable<V> implements Parcelable

A Parcelable that transfers data over a pipe created using createPipe. This circumvents the standard Android binder size limit allowing for passing objects of any size, limited only by the heap space of the sender and receiver process.

This class is not thread safe. To enable verbose logging: adb shell setprop log.tag.FdParcelable VERBOSE There are two copies of this file, one in clover-android-sdk and one in schema-tool, please keep them in sync.
