SessionContract

public class SessionContract

A contract class is a public final class that contains constant definitions for the URIs, column names, MIME types, and other meta-data that related to the provider. The class establishes a contract between the provider and other applications by ensuring that the provider can be correctly accessed even if there are changes to the actual values of URIs, column names, and so forth.
